• Öppna standarder innebär tekniker som är öppen för alla möjliga, dator, telefon osv.
  • Proprietära tekniker innebär tekniker som endast är öppen för vissa t.ex. på dator men inte telefon.
  • URL(Uniform Resource Locator) används som en adress för att lokalisera resurser.
  • URI(Uniform Resource Identifier) används för att identifiera resurser.
  • DNS(Domännamnssystemet) används för att koppla ihop rätt dömännamn med rätt IP-adress.
  • HTML(Hypertext Markup Language) är ett märkessrpåk för hypertext, med andra ord, ett språk som berättar hur innehållet av en websida ska se ut.
  • En slug är den unika delen av en URL och används för att vägleda var i websidan du är.
  • ASCII(American Standard Code for Information Interchange) är ett sätt att representera en vald uppsättning tecken t.ex. bokstäver.
